# PickZone

A community platform for racket sports players to track stats, join matches, and earn rewards.

## Stack

- Next.js 15
- React 19
- Tailwind CSS
- shadcn/ui

## Getting Started

`ash
yarn install
yarn dev
`

## Features

- Player dashboard with stats and leaderboard
- Signup flow with social login placeholders
- Modular data layer for stats and events
